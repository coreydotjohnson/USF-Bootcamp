Terminal Exercises
Write the following terminal commands to perform the following tasks:

Part I

make a directory called first mkdir first: mkdir first
change directory to the first folder: cd first
create a file called person.txt: touch person.txt
change the name of person.txt to another.txt: mv person.txt another.txt
make a copy of the another.txt file and call it copy.txt: cp another.txt copy.txt
remove the copy.txt file: rm copy.txt
make a copy of the first folder and call it second: cp -r first second
delete the second folder: rm -rf second

Part II

What does the man command do? Type in man rm. How do you scroll and get out?: man provides information on commands and their flags
Look at the man page for ls. What does the -l flag do? What does the -a flag do?: "ls -l" lists files with more detailed information. "ls -a" lists all files including hidden ones 
How do you jump between words in the terminal?: ctrl + left/right
How do you get to the end of a line in terminal?: ctrl + e
How do you move your cursor to the beginning in terminal?: ctrl + a 
How do you delete a word (without pressing backspace multiple times) in terminal?: ctrl + w
What is the difference between a terminal and shell?: shell processes commands. terminal gives access to shell.
What is an absolute path?: a full path that starts from the root
What is an relative path?: a path that starts from your current location
What is a flag? Give three examples of flags you have used.: a flag is an addition to a command that changes how it works. i've used -l, -a, -r
What do the r and f flags do with the rm command?: -r removes things recursively and -f forces the removal
